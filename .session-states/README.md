# Session States

This directory contains session state files from AI-assisted development sessions.

## Purpose

Session state files track:
- Development progress
- Decisions made
- Context for continuation
- Task completion status

## File Naming Convention

```
.session-state-YYYY-MM-DD[.md]
.session-state-YYYYMMDD[-description][.md]
```

Examples:
- `.session-state-2025-11-12`
- `.session-state-2025-11-12.md`
- `.session-state-20251112-refactoring.md`

## Usage

These files are:
- **Not committed** to git (in `.gitignore`)
- **Local only** - for personal tracking
- **Optional** - can be deleted if not needed
